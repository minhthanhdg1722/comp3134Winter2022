What is the definition of server port?
- A port is a number used to uniquely identify a transaction over a network by specifying both the host, and the service
What is server port range?
-Port numbers range from 0 to 65535
How many bits does number above represent?
-A port number is a 16-bit integer ranging from 0 to 65535

20: File Transfer Protocol (FTP)
